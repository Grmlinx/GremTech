# Gremoire Profile

## Archetype

Human-managed knowledge vault

## What To Preserve

- human-first readability
- simple vault structure
- clear promotion path from inbox to durable references

## Runtime Posture

Human-first and manual by default.
Background runtime behavior should be treated as an exception, not the norm.

## Default Review Lanes

- governance only when operating rules change
- product architecture for structure questions
- prompt engineering for AI startup-contract changes

## Standardization Focus

- keep the vault light
- preserve the distinction between personal notes and durable references
- use only the minimum governance needed to protect clarity
