# Governed Learning Loop

## Lesson

The agency should improve itself by promoting proven project lessons into
durable repo-owned surfaces instead of relying on recollection or prompt drift.

## Why It Matters

Without a governed learning loop, strong ideas are rediscovered repeatedly and
the repo becomes dependent on chat continuity.

## Reuse Rule

Future major projects should:
- inspect local analogues first
- inspect external references when useful
- capture inspiration scans
- promote reusable lessons into standards, templates, docs, or long-term memory

## Strong Promotion Targets

- `governance/standards/`
- `templates/`
- `docs/`
- `portfolio/`
- `memory/long-term/`
