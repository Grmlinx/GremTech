# Skills Before Agent Sprawl

## Lesson

Capability growth should prefer skill packs before new agents unless the
accountability boundary actually changes.

## Why It Matters

Without this rule, repos look sophisticated while the real orchestration and
decision-rights model gets noisier and harder to govern.

## Reuse Rule

When new repeated work appears:

1. ask whether it fits an existing accountability lane
2. add or improve a skill pack if it does
3. add a new agent only when escalation or decision rights truly differ
